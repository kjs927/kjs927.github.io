Installation of React on Ubuntu 18.04.4 LTS

A JavaScript library for building user interfaces.

  1. Install Node.js

  2. sudo npm install -g create-react-app

  3. npx create-react-app hello-app

  4. cd hello-app; npm start

  5. Done!
