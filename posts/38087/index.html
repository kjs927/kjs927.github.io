Building CI/CD Pipelines with IDEA, Gitlab, Jenkins and Kubernetes - Topic 1 - Gitlab

“Take your version control to the next level” - Gitlab.com

  1. Install Gitlab (Ubuntu 20.04 LTS)
    (1). sudo apt-get update

    (2). sudo apt-get install -y curl openssh-server

    (3). sudo apt-get install -y postfix

    • Quit the package configuration by selecting cancel in the Installing

    (4). sudo apt-get install -y curl openssh-server

    (5). Maybe there a problem on Ubuntu 20.04 LTS: [Unable to locate package gitlab-ee], so two logs here.

    • curl https://packages.gitlab.com/install/repositories/gitlab/gitlab-ee/script.deb.sh | sudo bash

      • sudo apt-get install gitlab-ee

    • curl https://packages.gitlab.com/install/repositories/gitlab/gitlab-ee/script.deb.sh | sudo os=ubuntu dist=bionic bash

      • sudo apt-get install gitlab-ee

    (6). sudo vi /etc/gitlab/gitlab.rb

    (7). sudo gitlab-ctl reconfigure
    (8). sudo gitlab-ctl restart

    Tips. useful commends (something went wrong while installing)

    (9). Done!
