Building a Spring Boot Application Using Intellij IDEA

‚ÄúSpring makes programming Java quicker, easier, and safer for everybody. Spring‚Äôs focus on speed, simplicity, and productivity has made it the world‚Äôs most popular Java framework.‚Äù - spring.io

  1. Install IntelliJ IDEA
  2. Start IntelliJ IDEA and Create New Project
  3. Select Spring Initializr -> Project SDK




    and Next.
  4. Project Settings (Optional)

    and Next.
  5. Select Dependencies (Enable Web -> Spring Web Here)

    and Next.
  6. Select Project Location
  7. Setup Maven (a build automation tool used primarily for Java projects)
    (1). File -> Settings

    (2). Build, Execution and Deployment -> Build Tools -> Maven

    (3). Download Apache Maven (https://maven.apache.org/) and Create a Folder for Maven Repository

    (4). Setup Apache Maven folder -> conf folder -> settings.xml


    (5). Fill Maven home directory, User settings file and Local repository in Intellij IDEA settings.

    Apply and OK
    (6). We Got Some Error Information from Default Maven Settings Here

    (7). Click Reload All Maven Projects on Top-Right Corner, and Wait for Maven Import Completion

    (8). Errors were resolved
  8. Create a Simple Web Application
  9. Click Run DemoApplication Button on Top-Right Corner (Shift + F10)
  10. Done!
