Building CI/CD Pipelines with IDEA, Gitlab, Jenkins and Kubernetes - Topic 2.5 - Jenkins (Another Plugings)

  • Email Notification (Gmail)
  • Code Check (Sonarqube)

  1. Email Notification
    (1). install Email Extension Templete


    (2). set up emaill notification (Gmail)

    • enter Configure System
    • find Extended Email Notification
    • get smtp settings from Gmail
    • get app password (if two-factor authentication is enabled)


    • fill Extended Email Notification form on Jenkins
    • find Jenkins Location and add admin email on Jenkins
    • fill Email Notification form and test email configuration on Jenkins
    • create email templete on project


    • reserved variables on Configure System->Content Token Reference->help button
    • commit and push

    • generate email script by syntax generator (similarly as before)

    • revise Jenkinsfile
    • commite and push

    • build test

    • done!

  2. Code Check

    (1). sudo apt-get install openjdk-11-jdk

    (2). sudo apt-get install postgresql

    (3). sudo passwd postgres

    (4). su - postgres

    (5). systemctl restart postgresql

    (6). wget https://binaries.sonarsource.com/Distribution/sonarqube/sonarqube-8.4.1.35646.zip

    (7). sudo apt-get install zip

    (8). sudo unzip sonarqube-8.4.1.35646.zip -d /opt

    (9). sudo mv /opt/sonarqube-8.4.1.35646/ /opt/sonarqube
    (10). sudo groupadd sonar
    (11). sudo useradd -c “SonarQube - User” -d /opt/sonarqube/ -g sonar sonar
    (12). sudo chown sonar:sonar /opt/sonarqube/ -R
    (13). sudo vim /opt/sonarqube/conf/sonar.properties

    (14). su sonar /opt/sonarqube/bin/linux-x86-64/sonar.sh start

    (15). trouble shooting: max virtual memory areas vm.max_map_count [65530] is too low, increase to at least [262144]

    (16). open sonarqube url 2.(13) sonar.web.port


    (17). login


    (18). install SonarQube Scanner


    (19). find SonarQube Scanner on Global Tool Configuration


    (20). add credentail for login sonarqube

    • get token from account settings


    • add sonarqube credentail


    (21). find SonarQube servers, fill and save it on Configure System

    (22). configure project

    (23). add sonar build script


    (24). build test


    (25). OK



    (26). add sonar-project.properties in project

    (27). move the script to sonar-project.properties

    (28). get variables name from Jenkins


    (29). add Code Check stage to Jenkinsfile on project

    (30). commit and push


    (31). build test


    (32). done!
